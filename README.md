# Installation


## This will clone the repository

```bash
git clone https://github.com/kryptt0/TrackIP
```

## Change into TrackIP directory

```bash
cd TrackIP
```
## Install Crips 

```bash
./install.sh 
```

## Permissions 

```bash
chmod ugo+rwx trackip
```

## FINISHED INSTALL
## FINISHED INSTALL


## Run Crips multitool 

```bash
python2 crips.py
```

## Run the trackip tool

```bash
./trackip
```
